# 2026-03-17 モバイル対応・スワイプ操作実装

## 対応内容

### レスポンシブ (< 768px)
- ナビゲーション: 横スクロール可能なピル型に変更（ハンバーガー不使用）
- ボタン: min-height 44px に拡大（Apple HIG 準拠）
- 入力フォーム: min-height 44px・フォントサイズ調整
- カードグリッド: 1カラムに変換
- セクション padding を縮小
- タッチデバイスでホバー浮上を無効化 (`@media (hover: hover)` で分離)

### スワイプ操作
- 左スワイプ → 次のセクションへスムーズスクロール
- 右スワイプ → 前のセクションへスムーズスクロール
- 縦スクロールと競合しない判定 (`|dx| > 60px && |dx| > |dy| * 1.5`)
- スワイプ時にセクション名 + ドットインジケーターが画面下部にフェードイン表示
- デスクトップでは非表示 (`@media (min-width: 768px)`)

### スクロールスパイ改善
- アクティブなナビリンクをモバイルで自動スクロール (`scrollIntoView`)
- インジケーターのドットとラベルをスクロールスパイと同期

## ファイル変更
- `src/style.css` — モバイルメディアクエリ・スワイプインジケーター CSS 追加
- `src/main.js` — スワイプ検出ロジック・インジケーター同期処理追加
- `index.html` — スワイプインジケーター HTML 追加
