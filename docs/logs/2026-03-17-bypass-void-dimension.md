# 2026-03-17 BYPASS → VOID次元 実装

## 世界観設計

- VOID SECTOR = ファイアウォール外側、登録されたネットワークの「狭間」
- 色: ウルトラバイオレット (#cc00ff) + 深紫黒 (#080010) — シアン世界との対比
- データが腐食・断片化し、座標・現実指数が崩壊していく演出

## トランジション（5段階）

1. BYPASSボタングリッチ (500ms) — 振動 + hue-rotate + brightness爆発
2. RGB分裂フラッシュ (650ms) — クロマティック収差、画面全体が色分離
3. スキャンワイプイン (500ms) — 横線で画面を「引き裂く」
4. VOIDコンテンツ出現 — 要素が順番にフェードイン (stagger)
5. JACK OUTで逆順復帰

## VOID内のライブエフェクト

- ホワイトノイズ Canvas (TV砂嵐、紫トーン)
- ヴォイドポータル — 3重リングが脈動
- NULL://DIMENSION グリッチタイトル
- 座標スクランブル (200ms間隔でランダム更新)
- 現実指数 decay (1.000 → 0.000 に向けて減衰)
- ログメッセージ 8行が順次出現
- 腐食データフラグメントが浮遊 (1.2秒ごとに生成)
